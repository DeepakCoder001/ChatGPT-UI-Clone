# 🤖 ChatGPT UI Clone

A simple ChatGPT-inspired web application built using **HTML, CSS, and JavaScript**. This project includes a ChatGPT-like interface with a basic rule-based chatbot. The chatbot responds to predefined commands and displays fixed replies based on the user's input.

---

## 🚀 Features

- ChatGPT-inspired user interface
- Simple chatbot using predefined responses
- Send messages using the Send button or Enter key
- Fixed replies for specific commands
- Responsive 

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome

## ▶️ How to Run

1. Clone the repository

- hi
